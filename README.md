# BioMed-Nexus

BioMed-Nexus is a React + Vite web app for exploring biomedical knowledge across genes, variants, compounds, clinical trials, and scientific literature.

## Features

- Search genes, variants, and drug names
- View linked genomic and molecular information
- Explore a knowledge-graph style network of biomedical entities
- Review clinical trials and PubMed literature results

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```
2. Start the development server
   ```bash
   npm run dev
   ```
3. Build for production
   ```bash
   npm run build
   ```

## Tech Stack

- React
- Vite
- CSS for the UI
